Find a Mechanic (In Progress Startup)

This is a side project I'm working with a startup to build. It's a web app and mobile app that people can use to book appointments with local auto repair shops. Features provide transparency regarding status of appointments and create an all-in-one environment for customers, garage owners, and employed mechanics. ChatStream

A web app where people can join public chat rooms quickly to meet new people and make friends. This app is different from others because it aims to make public chat groups easy to access for those looking to quickly find a conversation by showing all active chats right from the start.
